Wrangling and Chess Analysis

Data Source


This analysis focuses on extracting data from an unusual form and making usable datasets from them. Chess data comes in a '.pgn' file which is a specialized filetype used to record the information of any chess game. This project used publically available pgn files from Lichess.

Jump to analysis


If we open a pgn file (using a standard notepad opener), we see that the file can contain thousands of different games. Usually the first 17 or so lines of the file contain the metadata information of each game played (highlighted in blue). This metadata contains the names of the players, their ELO (ranking, higher is better), UTC date and time, their elo rating difference and a whole lot of other information. There then is a linebreak and the following line (highlighted in orange) contains info about the actual moves played and it is all contained within one single line.


So, we can view this .pgn file as a giant list of a singluar column data. Each row represents a different type of data [Date, time, round, name of black player, name of white player etc.] and each observation is a string which can be parsed through using string selectors. Finally each set of 19 lines contains data about one game of chess and any line that starts with the strings "[Event" means that the following rows contain data about another game.


So, now we need to separate the metadata from the actual moves played and create a dataset that takes all this "columnar list" data and makes it into a rectangular format; where each row is a different game played, and each column contains the metadata info and the actual moves played.

Essentially, this was using some clever string selection to first create variable names (in varnames) and using the similar method to create the values (in values) for the data that goes into each variable name. Finally, they're all merged together using the groupby() function alongside the pivot_wider() function.

Now lets start analyzing this data



How much ELO is playing first worth?

Lets create a linear model of the proportion of time the first player (white) wins as a function of the amount of higher rating the first player has over the second player. We'll use this model to determine how many rating points playing first is worth (i.e., how many rating points worse does the player using white have to be in order to win exactly half of the time).

# y = mx  + b ; 
# y = proportion of wins (Result = 1-0),
# x = Diff between positive WhiteElo - Black Elo 
# b = intercept

### ok let convert that resutls to 1 0 (White win or loss, count draws as loss)

# Create new data row that has a binary var of 1 or 0 based on white wins 
Result_binary <- c(nrow = nrow(newtest))

for (i in 1:nrow(newtest)){
  if (newtest$Result[i] == "1-0"){
    Result_binary[i] <- 1
  }  else {
    Result_binary[i] <- 0
  }
}

# create smaller df, with just elo vars and Result_binary for lm
Ratings_worth <- data.frame(WhiteElo = newtest$WhiteElo,BlackElo = newtest$BlackElo, Result_binary = Result_binary)

# filter where WhiteElo > BlackElo
Ratings_worth <- Ratings_worth[which((as.numeric(Ratings_worth$WhiteElo) > as.numeric(Ratings_worth$BlackElo))),]

# create WhiteDifference vector
Ratings_worth$WhiteDiff <- as.numeric(Ratings_worth$WhiteElo) - as.numeric(Ratings_worth$BlackElo)

# create the linear model 
Ratings_worth_mod <- lm(Result_binary~WhiteDiff, data = Ratings_worth)

summary(Ratings_worth_mod)

Chess openings and the popularity of e4

Out of the 20 possible moves that a player can make in chess, a very common on is the move e4. This is when the pawn on the e column (in front of the king) moves up two spaces. How popular is this first move compared to all other moves possible? Lets compare the popularity of e4 as the first move compared across elos.

# initialize first_move vector

So in the graph above we see that the popularity of e4 is relatively high across all the elo spectrum. Beginners and experts alike play e4 quite frequently. The sharp increase in the 850-900 elo bin shows that this group of players almost exclusively start e4. Once players start to become really good at the game (elo > 2000) the popularity of this start drops probably because they're now Masters or Grandmasters in the game and they know how to deal with this start or have more creative openings. It is still interesting to note that this one move out of the possible 20 moves occurs in about half of all chess matches.
